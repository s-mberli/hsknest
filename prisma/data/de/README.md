# German Vocabulary Data

All JSON files in this directory (`a1.json`, `colors.json`, `family.json`, `food.json`, `freq100.json`, `greetings.json`, `numbers.json`) are original, hand-authored works curated for the HSK Nest vocabulary trainer.
